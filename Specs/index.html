Original Design Specifactions

The original version of CitiTek has been designed under restrictions given by the CPU created by James Sharman. In his videos he goes into great detail of every stage of designing and making the CPU, from the clock to the NES controller inputs, you should seriously consider giving him a subscribe!

Features of the CPU

  • Sound: Four channel sound card allowing for 4-note polyphony.
  • SD Card support: An SD card running under fat-16 for the computer's seccondary storage running under SPI.
  • VGA Card: The VGA card can output 640x480 vga with 90x80 visible 8x8 pixel tiles.
  • Controller support: The CPU can use a NES controller for game input.
  • Fast speed: The CPU runs at 4.0mhz under normal running conditions.

Challenges

Composing the music for the game has been a chalange due to the fact there is only 4-note polyphony, which has made the trade-off between good music and sound effects difficult. In the end I have decided to keep it up to the user. They can simply switch between having music and sound effects, if I implement them.

Creating the artwork is not that easy either, as I am conformed to a grid of 8x8 tiles which does make positioning tricky! The mockups I have made for the screen and the like are looking great though! I eventually need to draw some buildings, which is not easy in isometric art! If you would like to, you can send your ideas for buildings on the Discord Server, and if they are cool I will add them to the game!

If you would like to contribute to the project, consider naming an NPC, and if you would like to watch the game progress, join the Discord server and join the community!
